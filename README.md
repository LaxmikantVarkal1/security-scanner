# aps - Security Scanner Dashboard

A modern, responsive, and beautiful dashboard interface built for a B2B SaaS security platform. It provides a clean view for managing security tasks, viewing live scan logs, and tracking system vulnerabilities.

## Features Built

- **Dashboard Screen**: A high-level overview of system status with a detailed, paginated data table for scans. Includes search, status filtering, and column show/hide capabilities.
- **Scan Detail Screen**: An interactive terminal-like console that simulates real-time scan progress, alongside a detailed finding log highlighting vulnerabilities by severity.
- **Custom Theming**: Full support for both beautiful Light and sleek Dark modes.
- **Polished UI**: Uses smooth skeleton loaders, transitions, hover states, and dynamic status badges to deliver a premium user experience.
- **Responsive Navigation**: A collapsible sidebar specifically designed to work perfectly on both desktop and mobile devices.

## Tech Stack Used

This project was built strictly focusing on modern web standards and high performance:

*   **React 19**: The core framework for building the user interface.
*   **Vite**: A lightning-fast development server and build tool.
*   **TypeScript**: Used throughout the application to ensure type safety and catch errors early.
*   **Tailwind CSS (v4)**: For rapid, utility-first styling without ever having to write custom CSS files.
*   **React Router**: For seamless navigation between the Dashboard and Scan Detail screens.
*   **Lucide React**: For crisp, customizable SVG icons.
*   **Radix UI / Shadcn**: For accessible, unstyled foundational UI components (like the sliding Drawers).
*   **Sonner**: For beautiful, modern toast notifications.

## Getting Started

To run this project on your own machine:

1. **Install dependencies:**
   `npm install`

2. **Start the development server:**
   `npm run dev`

3. **Build for production:**
   `npm run build`
